# C - Hash tables

## Description
This project introduces the concept of **hash tables**, a powerful data structure that allows fast data retrieval.  
You will implement basic operations such as creating a hash table, storing key/value pairs, retrieving values, printing the table, and deleting it.

By completing this project, you will understand how hash tables work internally and why they are widely used in many programming languagesâ€”such as Python dictionaries.

## Learning Objectives
At the end of this project, you should be able to explain:

### General
- What is a **hash function**
- What makes a **good** hash function
- What is a **hash table**, how it works, and common use cases
- What is a **collision** and its handling techniques (Chaining)
- The advantages and limitations of hash tables
- How to implement basic hash table operations in C
